# infix-conversion# Infix Expression Conversion

## Description
This project converts infix expressions into postfix and prefix using Java.

## Features
- Infix to Postfix conversion
- Infix to Prefix conversion
- Uses stack data structure

## Example
Input: (A+B)*C  
Postfix: AB+C*  
Prefix: *+ABC  

## How to Run
1. Compile:
   javac InfixConversion.java

2. Run:
   java InfixConversion
